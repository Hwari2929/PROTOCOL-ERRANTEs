# PROTOCOL ERRANTES — 디자인 시스템

> 매너: **세피아 빈티지 · DIY 펑크 · 종이/영수증 감성 · 그레인 텍스처.**
> 깨끗한 인쇄물/등사 전단지/현장 서류철처럼. 광택·3D·네온 금지.

## 1. 팔레트 (sepia paper)

| 역할 | 색 | RGB |
|------|-----|-----|
| 종이 베이스 | 크림 | `Color(0.90, 0.85, 0.74)` |
| 패널/카드 | 밝은 종이 | `Color(0.93, 0.88, 0.78)` |
| 잉크(본문/테두리) | 짙은 갈흑 | `Color(0.16, 0.12, 0.08)` |
| 잉크 연함(보조) | 세피아 | `Color(0.36, 0.30, 0.22)` / `#5c4d39` |
| 스탬프 레드(강조/위험) | 빛바랜 빨강 | `Color(0.62, 0.23, 0.18)` |
| 세피아 골드(등급/메커니즘) | `#8a662a` |
| 올리브(버프/상태) | `#4d6a30` |

본문 텍스트는 **항상 짙은 잉크색** (밝은 종이 위 어두운 글씨 = 인쇄물 원칙).

## 2. 타이포그래피

- **본문**: `KoPubBatang-Medium` (명조/세리프 — 인쇄물 느낌). 테마 default_font.
- **헤더/강조(펑크 포스터)**: `Paperlogy-8ExtraBold` / `9Black`. 스탬프 레드.
- **수치/영수증 라인**: 본문 명조 사용 (별도 모노 폰트 없음).

## 3. 패널·버튼 매너 (영수증/종이 카드)

- **모서리 직각** (corner_radius 0) — 종이 잘린 느낌.
- 테두리: **잉크색 2px 실선**.
- 버튼 눌림: **잉크 채움 + 종이색 글씨** (스탬프 찍힌 반전).
- 호버: 종이 약간 어둡게 + 테두리 스탬프 레드.
- 구분선이 필요하면 점선/파선 잉크 룰 (draw 코드로).
- 이 규칙은 `assets/theme/main_theme.tres`에 박혀 있어 모든 Control이 상속.
  코드에서 패널이 필요하면 **`Panel` 노드 사용**(테마 자동 적용). raw ColorRect는 종이색만.

## 4. 텍스처 처리

- 배경: `scripts/ui/background.gd` (CanvasLayer −10) = `assets/ui/paper.png` 종이 베이스
  + 절차적 **필름 그레인**(FastNoiseLite, 낮은 불투명도) 오버레이.
- **3D/사실적 배경 그래픽은 숨김** (assets/bg/* 미사용). 분위기는 종이+그레인으로만.

## 5. 새 UI를 추가할 때

1. 색은 위 팔레트에서만 고른다. 임의 색 금지.
2. 버튼/패널은 `Button`/`Panel` 노드 → 테마 자동. 폰트 사이즈만 오버라이드.
3. 직접 그리는 요소(_draw)는 잉크/세피아/스탬프 레드로. 종이 위 어두운 잉크 원칙.
4. RichTextLabel은 `default_color`를 잉크로 오버라이드 후 bbcode `[color=#5c4d39]`(세피아)/`#8a662a`(골드)/`#4d6a30`(올리브)/스탬프 레드만 사용.
